# CarParkJS

Example NodeJS/JavaScript (ES2018) project with classes and Jest-tests.

Requirments:

- Node / NPM (v. 10 or newer)

Setup:

```
$ npm install
```

Running the application:

```
$ node main
```

will output after 10 seconds:


The application is tested with Jest:

```

```
